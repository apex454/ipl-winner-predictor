# 🏏 IPL Match Prediction using Machine Learning

This project focuses on predicting the outcome of IPL matches using basic match-level data such as teams, toss results, and venue.

## 🚀 What I Did

- Collected IPL match data and selected important features like:
  - Team names  
  - Toss winner  
  - Match venue  
- Converted the prediction problem into a binary classification task  
- Trained a machine learning model to predict match outcomes  

## 🤖 Model Used

- Random Forest Classifier

## 📊 Accuracy

The model achieves around **50–60% accuracy**, which is reasonable since cricket matches are highly unpredictable.

## 📂 Dataset

- IPL match dataset (team, toss, venue, winner)
